Prepare Docker Buildx
-----

Used to make sure `docker buildx` can be used on Linux Docker Digital Ocean Droplets built late 2021.

Usage
-----

    export DOCKER_CLI_EXPERIMENTAL=enabled && ./scripts/run.sh

Resources
